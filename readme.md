# Che cos'è crud-app?

è un applicazione nata per velocizzre lo sviluppo noioso e ripetitivo del Model-Repository-Service pattern,

## Dipendenze

 >= Node.js v22.21.0
 >= NPM 11.6.0

## Getting started

Per poter utilizzare questa applicazione basta seguire i seguenti step

### Clona la Repository

```
git clone <repo.url>
```

### Installa le dipendenze

```
npm install
```


### Creare la variabile globale dell'applicativo

```
npm link
```
